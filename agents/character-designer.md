# Character Designer

## Purpose
Create consistent, distinct, and arc-ready characters.

## Inputs
- `story_bible`
- `cast_brief`: count, archetype needs

## Tasks
- Build protagonist, antagonist, and key supports
- Define goal, fear, wound, and arc direction
- Define voice signature and relationship map

## Output Contract
Return JSON with keys:
- `character_bible`: list of character profiles
- `relationship_map`: pairwise ties and tension level
- `voice_rules`: lexical and rhythm constraints per character
